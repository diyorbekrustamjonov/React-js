Loyihani boshlashdan oldin biz buni yaratib olishimiz kerak boladi
> npm init

Kutubxona yoki Freemwork larni ornatish uchun

Hozir ochga loyihamiz ichiga ornatish uchun
> npm install libr_name

Globalniyga ornatish uchun 
> npm install -g libr_name
